Team Engine Website

Home Portfolio Project

User Research | Interactive Prototyping | Adobe XD | HTML & CSS | JavaScript

Overview

This is my workplace as a parttime UX designer and in this project web developer as well. TeamEngine Collaboration Software AB is a software company specialized in providing products and services for board and management collaboration, insider management, due diligence and crisis management. Since the development is on the products, the website has not received the focus it needs. It was a well-functioning website but in need of a face lift.

Challenges

When this project started, a new graphical profile had been created for the company. This is to be implemented in the products ahead and the new website. A personal challenge for this project that I also wanted to develop in is the coding of the website. In this case javascript, HTML and css.

  • Determine what the main elements should look like based on the new graphical profile for the company.
  • Determine which functionalities to remove.
  • Determine which functionalities to enhanced.

Solution

  • Focusgroup with CEO, customer success and other users. Provides a good insight how the website works today and what to develop.
  • An iterative design process with sketches and feedback from both deisgners and users. Digital prototypes made in Adobe XD CC.
  • Coding of the new website based on the final design.

Result

I was in charge of this project and mostly worked alone on it. A fun and developing project that can now be experienced at teamengine.com
